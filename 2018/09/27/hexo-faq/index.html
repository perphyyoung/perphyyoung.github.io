2. 添加 favicon.ico

问题：
默认情况下，在网页的标签栏显示的 favicon.ico 为空白。

解决：

  1. 制作 ico 图标，建议尺寸 48*48，命名为 favicon.icon

  2. 将图标置于 blog/source 目录下，修改 blog/_config.yml

  3. 在本地应该是可以马上呈现，在 GitHub 网址上比较慢，耐心等待

参考：

  1. Hexo搭建的个人博客，怎么显示网站的 favicon.ico？
  2. 在线生成透明 ICO 图标
